# 觉醒
必须要等级为120级时才能觉醒

#### 觉醒
解放技能等级8,物理攻击有几率暴击(%1),暴击时攻击力上升至150%,而且击倒敌人.

#### 超觉醒
解放技能等级9,可以被动闪避敌人的攻击(%1)

#### 终极觉醒
解放技能等级10,可以使用终望核心
