# 近身型武器

主要是近战攻击使用的武器;

#### 剑(Sword):
* 武器系:剑系武器
* 普通攻击属性:斩击
* 武器类型:单手或双手
* 简介:属性适中,全能型的近战武器

#### 阔刃剑(Broadsword);
* 武器系:剑系武器
* 普通攻击属性:斩击
* 武器类型:双手
* 简介:攻击力较高,防御力较高,速度慢

#### 长剑(Long Sword):
* 武器系:剑系武器
* 普通攻击属性:斩击
* 武器类型:双手
* 简介:攻击力高,速度慢

#### 巨剑(Great Sword):
