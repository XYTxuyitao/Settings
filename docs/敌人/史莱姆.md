# 史莱姆

#### 小型史莱姆

编号:SL01

生命:20

防御:5

攻击:5

行动值:5

技能:

* 撞击:对一个目标造成 **100%** 的打击伤害,消耗2点行动值
* 舔舔:对消耗一个目标 **1**点行动值,自己消耗2点行动值

特殊(复仇):

* 腐蚀:被小型史莱姆攻击的目标会每回合扣 **4** 点生命值
