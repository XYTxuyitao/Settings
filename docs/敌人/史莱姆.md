# 史莱姆

---

#### 小型史莱姆

编号:SL01

生命:15

防御:5

攻击:5

行动值:5

技能:

* 撞击:对一个目标造成 **100%** 的打击伤害,消耗2点行动值

特殊(复仇):

* 腐蚀:被小型史莱姆攻击的目标会每回合扣 **4** 点生命值

#### 中型史莱姆

编号:SL02

生命:25

防御:8

攻击:12

行动值:6

技能:

* 撞击:对一个目标造成 **100%** 的打击伤害,消耗2点行动值

特殊(复仇):

* 高级腐蚀:被小型史莱姆攻击的目标会每回合扣 **6** 点生命值
* 分裂:当生命小于12时,下一回合分裂成两个 **小型史莱姆**  小型史莱姆的生命值等于中型史莱姆剩余的生命值
